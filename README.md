# minitalk (42 Cursus)
Communicate between two programs (client and server) using UNIX signals.

# Compilation
``` bash
$ git clone https://github.com/azfarjef/minitalk.git amini
$ cd amini && make
```
run server
``` bash
$ ./server
```
send message from client to server
``` bash
$ ./client [server-pid] [message]
```
